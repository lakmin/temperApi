# temper

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
1 Backend (VANILA PHP)
i. cd api
ii. php -S localhost:1000
```

2. Backend (Laravel)

i. cd api_laravel
ii. composer install
iii. npm install
iv. create a new database and edit .env file on the root 
i. php artisan migrate
vi. php artisan serve
```
3 Frontend
i. npm run serve
```
